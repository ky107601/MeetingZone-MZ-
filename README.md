# MeetingZone-MZ-

**MZ** 는 다중 영상 채팅 프로그램으로, 자신의 영상을 원하는 좌표로 이동 가능한 것이 특징이다. 

  
## :clock2:개발 기간
24.11.11 ~ 24.12.13

  
## 👨‍👩‍👦‍👦팀원
김시후, 김연경, 김유진, 김한중, 이재용
